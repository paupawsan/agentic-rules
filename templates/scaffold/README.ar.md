4. فعل المكون الإضافي باستخدام `python setup.py`

## التكوين

### الإعدادات الأساسية (`settings.json`)

```json
{
  "{{plugin_key}}": {
    "enabled": true,
    "config": {
      "example_setting": "example_value",
      "max_entries": 100,
      "cleanup_days": 90
    },
    "advanced": {
      "debug_mode": false,
      "performance_mode": "balanced"
    }
  }
}
```

### وصف الإعدادات

- `enabled`: تفعيل/تعطيل المكون الإضافي
- `config.max_entries`: الحد الأقصى لعدد الإدخالات المراد الحفاظ عليها
- `config.cleanup_days`: أيام الاحتفاظ بالبيانات قبل التنظيف
- `advanced.debug_mode`: تفعيل تسجيل التصحيح
- `advanced.performance_mode`: وضع تحسين الأداء

## الاستخدام

1. **تفعيل المكون الإضافي**: اضبط `enabled: true` في `settings.json`
2. **تفعيل القواعد**: استخدم `python setup.py` لإنشاء ملفات القواعد
3. **التكامل**: انسخ ملف القواعد المُنشأ (مثل `AGENTS.md`) إلى مشروعك
4. **التكوين**: قم بتخصيص الإعدادات في `{{plugin_name}}/settings.json`
