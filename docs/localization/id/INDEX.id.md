## Panduan Cepat

### Untuk Pemula
1. Mulai dengan [../../README.id.md](../../README.id.md) untuk penjelasan singkat
2. Baca [../../CORE-RULES.id.md](../../CORE-RULES.id.md) untuk memahami aturan dasar
3. Periksa bagian khusus platform di README

### Untuk Programmer
1. Lihat [EXTENSION-MANUAL.id.md](EXTENSION-MANUAL.id.md) untuk menambahkan aturan baru
2. Pelajari bagaimana aturan yang ada dibuat
3. Ikuti daftar periksa untuk penambahan fitur

### Untuk Pengintegrasi
1. Buka [`../bootstrap.json`](../bootstrap.json) untuk pengaturan teknis
2. Atur [`../settings/global-settings.json`](../settings/global-settings.json)
3. Sesuaikan pengaturan platform sesuai kebutuhan

## Tujuan Dokumentasi

- **Dokumentasi framework** (README, CORE-RULES) dibuat agar manusia mudah memahami sistem
- **Dokumentasi aturan** (file RULES.md) berisi spesifikasi teknis untuk AI
- **Konfigurasi** (file JSON) adalah pengaturan yang bisa dibaca komputer
- **Manual ekstensi** memberikan panduan langkah demi langkah untuk pengembangan

## Berkontribusi pada Dokumentasi

Saat menambahkan aturan atau fitur baru:

1. Perbarui file dokumentasi yang relevan
2. Tambahkan entri ke daftar isi ini
3. Ikuti format dan struktur yang sudah ditentukan
4. Sertakan contoh dan cara mengatasi masalah

## Versi
Dokumentasi ini untuk versi framework 1.0.0.
